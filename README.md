# 教学过程资料归档管理系统

## 项目简介

这是我们小组这学期Web前端课的项目作业，
**基本任务**是实现教学过程资料（包括作业、随堂测验、课堂汇报、考试卷面、成绩等）的整理和归档，
涉及的用户包括任课教师、教学干事以及校领导。

其中，任课教师可以上传、整理和查阅资料；教学干事可以审阅、分类和反馈教师上传的资料，以决定其是否能被归档；而校领导可以查看归档的资料。
